## What is cloud?
Cloud is a network of remote servers hosted on the internet that store data, run applications, and deliver computing services instead of using a local computer or on-premise servers.

These servers are hosted in large data centers managed by cloud providers such as:

- Amazon Web Services
-  Microsoft Azure
- Google Cloud

Instead of buying hardware, users rent computing resources on demand.

Example:
Instead of storing photos on your laptop:

You upload them to cloud storage, and you can access them from:
Laptop
Mobile
Tablet
Anywhere with internet


## What is Cloud Compuing?

Cloud Computing is the on-demand delivery of computing services such as servers, storage, databases, networking, and software over the internet with pay-as-you-go pricing.

This means:
- No need to buy servers
- No need to maintain hardware
- Resources can scale automatically
- You pay only for what you use

## Key Characteristics of Cloud Computing

These 5 characteristics are very important 
* On-Demand Self Service
    Users can create servers or resources whenever needed.
* Broad Network Access
    Services can be accessed from anywhere through the internet.
* Resource Pooling
    Cloud providers share resources among multiple customers.
* Rapid Elasticity
    Resources can scale up or down quickly.
* Measured Service
    You pay only for the resources you consume.
